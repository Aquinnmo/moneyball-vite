# Moneyball Front End

This project is a simple front end to display advanced MLB metrics fetched from the backend API. The front end should have to do minimal to no calculations. The heavy lifting will be done in the front end (NOT contained in this repo).

## Tech Stack

React + Typescript
React Router for routing

## Data

Fetched from the official MLB API and baseball savant endpoints, then cleaned for the front end through a Spring Boot server.
