# Moneyball: Project Jarvis

A sci-fi inspired, data-centric front end designed to visualize advanced MLB metrics as a high-fidelity holographic HUD.

## Visual Direction
This project utilizes the **Hologram Theme**, focusing on:
- **Semi-Transparency**: Layered interfaces using navy depths.
- **Glow-Heavy UI**: High-contrast cyan accents with neon-like luminosity.
- **Projector FX**: Integrated scanlines and subtle flicker animations for a "Jarvis" style experience.

## Tech Stack

React + TypeScript + Vite
React Router for routing
Hologram-spec CSS (Monospaced, Data-first)

## Data

Fetched from the official MLB API and baseball savant endpoints, then cleaned for the front end through a Spring Boot server.
